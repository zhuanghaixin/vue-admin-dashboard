# vue-admin-dashboard
基于Vue.js开发的仪表盘
[在线体验地址](https://zhuanghaixin-vue-admin-dashboard.netlify.com/)
## 技术栈
```
Vue.js+Vue-Router+Vuex+Sass
```

## 项目介绍
此项目是作为Vue全家桶的练习，实现登录，注册，仪表盘，图表功能等，用Netlify进行部署
## 业务实现
```
* 登录
* 注册
* 忘记密码
* 换肤
* 仪表盘状态展示
* 图表展示
```
## 交互实现
* 全局换肤
* 状态展示
* 基本的弹出框交互
